# BlendDistance Shader for ME
A shader mod for HS2 / AIS Studio, based on the shader for Unity Editor, [Blake/BlendDistance](https://github.com/Blatke/BlendDistance.shader).

This mod is to make the object's material transparent at its edges beside another object. It can be applied in some circumstance such like clouds, fogs or semen...

## Installation
Go to the [**Release**](https://github.com/Blatke/BlendDistance-Shader-for-ME/releases) page, and download the **.zipmod** file for the latest version. Drag and drop this .zipmod file into the **/mods/** folder in your game directory. Then, start your Studio.

## How to Use
In Studio, select your object on the Workspace tab, and click the MaterialEditor (ME) button to open ME tab. On this newly opened tab, on which you can edit the material(s) of your selected object, find the option namely "_Shader_", and click it to open the pop-up menu. Look for **Blake/BlendDistance** on the menu and then click it. Then the material of the object is using this new shader.
